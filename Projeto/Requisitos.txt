* Página com o conteúdo do silo *
Todas as informações para inserir devem ser inseridas em uma página para cadastro do silo.
E na página do conteúdo do silo, deve ser possível edita-las.

->  Painel de avisos para possibilidades de aeração.
as possibilidades são : Por convervação, por secagem,por semi-auto, por emergência.
No painel de avisos deve conter: input (String) para "equilíbrio" higroscopico e atual.

->  Painel com o produto armazenado.
Deve ser possível selecionar o produto sendo eles : arroz, milho, soja, sorgo, trigo e cevada.
Deve mostrar o produto selecionado com uma foto de indentificação.
Deve ser possível inserir a variedade (String).
Deve ser possível inserir a safra (String).

-> Painel com legenda das cores.
As core devem ser configuradas para o usuário, as possibilidades são : Curto, Aberto,
Não lido pelo chaveador, sem nível de produto, gradiente de temperatura, Sensor com erro.
obs:Gradiente de temperatura pode conter duas cores para setar.
